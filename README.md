# Current Project

## 🔥 Phoenix

Phoenix is an open-source research platform for analyzing Solana markets.

Instead of predicting prices, Phoenix focuses on identifying manipulation before users engage with a token.

Current research includes:

- On-chain fee analysis
- Wash trading detection
- Market classification
- Behavioral pattern recognition
- Real-time telemetry
- Defensive tooling

The goal isn't to generate buy signals.

The goal is to help people understand **what they're actually looking at.**

# What I've Learned

The biggest lesson I've learned wasn't technical.

It was realizing that software isn't valuable because it produces output.

It's valuable because the output can be trusted.

After building the first version of Phoenix, I realized the architecture had become difficult to reason about.

Instead of continuing to patch it, I rebuilt the system with stronger module boundaries, clearer responsibilities, and a focus on verification.

That experience completely changed how I think about engineering.
